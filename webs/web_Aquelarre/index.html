❓ Ayuda — Guía de botones

Pantalla de inicio

  • ➤ ENTRAR A LA BIBLIOTECA — Oculta la pantalla de bienvenida y abre la aplicación.

Barra de pestañas

  • 📖 Biblioteca — Muestra todos los manuscritos catalogados.
  • 📜 Manuales — Filtra solo los manuales básicos.
  • ⚔ Expansiones — Filtra las expansiones oficiales.
  • 🗺 Módulos — Filtra las aventuras y módulos.
  • 📜 Suplementos — Filtra los suplementos.
  • 🌍 Mapas — Filtra los mapas.
  • 📚 Rolmasters — Muestra la revista Rolmasters en un iframe.
  • ❓ Ayuda — Muestra esta guía y la información del programa.
  • ⚙ Configuración — Abre el panel de ajustes (tema, tamaño, resets).

Barra de controles

  • 🔍 Buscar — Escribe para filtrar por título, descripción o cualquier texto.
  • ★ Favoritos — Muestra solo los marcados como favoritos.
  • ✓ Leídos — Muestra solo los marcados como leídos.
  • 📖 Todos / tipo — Filtra por tipo de manuscrito (manual, aventura, mapa…).
  • ↑ / ↓ — Cambia entre orden ascendente y descendente.
  • 📂 / 📄 — Alterna entre el modo carpetas (árbol) y el modo plano.
  • ⊞ / ☷ — Alterna entre la vista cuadrícula y la vista lista.

Panel lateral

  • ☰ Navegar categorías — Abre o cierra el árbol de categorías para explorar por carpetas.

Botones de cada tarjeta

  • ♥ / ♡ — Marca o desmarca el manuscrito como favorito.
  • ⊙ / ✓ — Marca el manuscrito como leído o no leído.
  • 📋 — Abre el panel de detalle con información completa del manuscrito.
  • 📁 — Copia la ruta de la carpeta al portapapeles.
  • 📄 — Abre el archivo PDF directamente.

Panel de detalle (al hacer clic en 📋)

  • ✕ — Cierra el panel de detalle.
  • 📄 Abrir PDF — Abre el archivo PDF.
  • 📁 Carpeta — Copia la ruta de la carpeta al portapapeles.
  • ♥ / ♡ — Marca o desmarca como favorito.
  • ⊙ No leído / ✓ Leído — Alterna el estado de lectura.

Configuración (⚙)

  • ☆ Reset Favoritos — Borra todos los favoritos.
  • ✓ Reset Leídos — Borra el estado de lectura de todos los manuscritos.
  • ⭐ Reset Valoraciones — Borra todas las valoraciones por estrellas.
  • 📝 Reset Notas — Borra todas las notas personales.
  • ⚠ Reset Todo — Borra todos los datos guardados (requiere doble confirmación).

💾 Importar / Exportar datos

Tus valoraciones, favoritos, notas, estado de lectura y ajustes de visualización se guardan en el navegador. Con ⬇ Exportar datos puedes descargar un archivo .json con toda esa información para llevarla a otro ordenador o mantener una copia de seguridad.

Para restaurar una copia, usa ⬆ Importar datos desde el mismo panel de Configuración, o bien arrastra el archivo .json directamente a la pantalla de bienvenida al abrir la web. Si es tu primera vez y no hay datos guardados, verás una zona de destino señalada en la portada.

Consejo: exporta regularmente si usas la biblioteca desde varios dispositivos o navegadores. El archivo exportado contiene todo lo necesario para restaurar tu estado exacto.

💾 Persistencia de datos

Tus valoraciones, favoritos, estado de lectura y notas se guardan automáticamente en el navegador (localStorage). No se pierden al cerrar la página.

🔍 Búsqueda y filtros

Escribe en el campo de búsqueda para filtrar por título, descripción o cualquier texto. Usa los botones ★ Favoritos, ✓ Leídos para filtrar por estado, y el desplegable de tipos para filtrar por categoría. Cambia el orden con el menú desplegable y el botón ↑↓.
